# vietthan.github.io

# 1. Install & Run Locally

1. Install your venv
```shell
python3.11 -m venv venv
```
2. Install the dependencies
```shell
venv/bin/python -m pip install -r requirements.txt
```
3. Run locally
```shell
venv/bin/mkdocs serve
```
